# React Front End

### This front end is designed to allow users to upload COBOL code for analysis. The front end is built using React and TypeScript, and utilises IBM Carbon components for visual elements. The goal of this project is to provide explainability of L&Gs COBOL code, making it easier for developers to understand, explain and maintain.

## Getting started

### Setup

1. Run `npm install` to install all of the required dependencies.
2. Add requred environment variables in `.env` file.
3. Run `npm run start` to start development server.

Features
--------

The following features are available in this front end:

### Upload COBOL Code

Users can upload COBOL code for analysis. The code is processed and displayed in a readable format, allowing developers to easily understand the code's functionality.

### Explainability

The front end provides explainability for the COBOL code, making it easier for developers to understand and maintain. This includes information such as function calls.
